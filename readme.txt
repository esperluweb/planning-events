A simple and modern event planner plugin. Add events and display them anywhere using a shortcode.

== Description ==

Planning Events is a lightweight plugin to manage and display upcoming events in a clean, modern list format.  
Easily add events from the admin interface, and use the shortcode `[planning_events]` to display them anywhere on your site.

= Features =

* Create and manage events with date, time, location, and description
* Display upcoming events with a customizable shortcode
* Responsive design for all devices
* Intuitive admin interface

== Installation ==

1. Upload the `planning-events` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Start adding events via the **Planning Events** admin menu.

== Usage ==

Use the shortcode `[planning_events]` in any page or post.

Optional parameters:

* `limit` — Number of events to display (default: -1 for all)
* `order` — Sort order (ASC for ascending, DESC for descending — default: ASC)

Example:
`[planning_events limit="5" order="ASC"]`

== Frequently Asked Questions ==

= Is the plugin responsive? =  
Yes. The event list adapts to any screen size.

= Can I use the shortcode multiple times? =  
Yes. You can place it in as many pages or posts as needed.

= Can I customize the design? =  
Yes. You can override the plugin CSS from your theme or use the WordPress Customizer.
